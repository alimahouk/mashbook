Imagine the equivalent of tabbed browsing but for your AI chats.

Highlight any snippet within a chat and instantly branch off to a new chat with the full context of the previous chat, without disrupting the flow of your current chat.

Try it out! (no registration required)

1. Talk about any topic you can think of.

2. If you want to go into detail about something specific within the chat without going off-topic in the current chat, highlight that part and ask Mashbook about it.

3. You'll branch off into a new chat about that snippet with the full context of the existing chat. Now you can go in depth on this new topic without cluttering up the other chat!

4. The snippet you highlighted earlier is now linked to the new branch of the chat you just created, allowing you to return to it within the flow of the other chat.

5. You can also access all your chats at any time from the main chat list.
